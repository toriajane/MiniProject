What are CSS animations?

As we mentioned before; animations allow you to put an element in movement by allowing the element to gradually change from one style to another. Some of the properties to make animations are:

  • @keyframes
  • animation-duration
  • animation-iteration-count
  • animation-direction

Animation-duration

It is used to determine how long an animation should last when it's applied to an element, like a moving image or text.

Animation-iteration-count

It is used to indicate how many times you want an animation to repeat.

Animation-direction

It controls the direction in which an animation plays. It determines if the animation runs forwards, backwards, or alternates between forwards and backwards.

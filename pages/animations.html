What are CSS animations?

As we mentioned before; animations allow you to put an element in movement by allowing the element to gradually change from one style to another. Some of the properties to make animations are:

  • @keyframes
  • animation-duration
  • animation-iteration-count Remember to always set this property since its default duration is 0 seconds!!!!
  • animation-direction

What is CSS Syntax?

CSS Syntax refers to the set of rules for writing CSS.

A CSS rule is comprised of a selector, a property, and a value:

  1. The selector determines the HTML element that is being targeted for the following CSS styles, and is followed by an opening curly brace which marks the beginning of the CSS code.

  2. The property represents the specific characteristic of the HTML element which is being altered.

  3. The value is separated by the property by a colon and defines how the element should be styled.

What is CSS Flexbox?

CSS Flexbox is a one-dimensional layout method used to easily adjust the size and position of items in order to create a flexible and responsive design. A one-dimensional layout refers to laying out items in a row or column.

display

The display property is used to determine how the element is displayed. The default display value for most elements is either block or inline. However, if you set the value to flex, items will be positioned according to the Flexbox layout method:

display: flex

justify-content

The justify-content property is used to specify how the browser distributes space between and around items along the main axis of a flex container.
A few important values for this property include:

  1. space-around
  2. space-evenly
  3. space-between

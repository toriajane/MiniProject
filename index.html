Syntax

CSS Syntax referes as how we communicate with the browser, which is via Selectors, Properties and Values, they allow us to style a webpage with different details such as adding color, font type and font size for example.

Flexbox

Flexbox is a layout system that makes life easier since it allows you to design and build responsive web pages

Animations

Animations in CSS allow you to put an element in movement by allowing the element to gradually change from one style to another, we will help you understand what CSS animations are and how they work.

Transitions

Transitions in CSS enable you to make smooth changes to the values of certain properties over a specified period of time, we will help you understand what CSS transitions are and how they work.
